Latar Belakang

Coding Smart School CSS merupakan lembaga khusus MercySmart Homeschooling di bawah Yayasan Pendidikan Hasian dengan Akta Notaris Tahun 2000 yang diperbarui dengan Akta Notaris Gerda Joice Lucia Tahun 2013.

Gagasan mendirikan Coding Smart School CSS dimatangkan sejak Tim MercySmart Homeschooling menjadi Finalis ajang Hackathon Merdeka di kompleks Istana Negara Jakarta, Agustus 2015. Dibandingkan dengan 80 tim peserta lainnya (yang semuanya lulusan Perguruan Tinggi dalam dan luar negeri dan berpengalaman tahunan menjadi programmer) maka Tim MercySmart yang terdiri dari seorang ibu rumah tangga dan ketiga siswanya 11 tahun, 15 tahun, dan 16 tahun menjadi fenomena istimewa saat itu.

Bahkan 4 menteri yang hadir di acara tersebut menunjukkan apresiasinya bagi Tim MercySmart Homeschooling, khususnya Andre Christoga (11 tahun).

Bahwa keterampilan coding bisa dipelajari sedini mungkin, tidak perlu tunggu kuliah, tidak harus jadi sarjana. Sebagaimana di berbagai negara maju, coding sudah dipelajari sejak elementary school, tetapi mengacu kondisi pendidikan Indonesia sekarang, semestinya Coding sudah dipelajari dan dikuasai semasa SMA bahkan sebelum SMA.. Bahwa Coding bisa dipelajari siapa saja, termasuk ibu rumah tangga.

Coding Smart School CSS adalah lembaga pendidikan setara SMA khusus untuk mendidik siswanya secara full-timer dan part-timer di bidang coding computer. Coding Smart School CSS segera membuka Homeschooling setara SMA dengan fokus belajar coding dari Senin sd Jumat. Dan pada tahun ajaran baru 2016/2017, Coding Smart School CSS dengan sistem flipped-up class-room dibuka di Grand Emerald Kelapa Gading Jakarta dan Flavour Bliss Alam Sutera, Banten.
