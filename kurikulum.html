Kurikulum Coding Smart School

Semua materi pembelajaran baik materi Kemdikbud maupun Coding bertujuan untuk mempersiapkan siswa dan lulusannya mempunyai Skill Abad 21.

Skill Abad 21 adalah keterampilan untuk mampu bersaing, bertahan hidup di era kompetisi global dan ekonomi, berbasis pengetahuan dan kecakapan yakni:

Kemampuan Bekerjasama
Kemampuan Berkomunikasi secara Efektif
Kemampuan Berpikir Kritis
Kemampuan Pemecahan Maslah
Kemampuan Pengarahan Diri
Dan Kemampuan Literasi Media dan Teknologi

Berbeda dari hasil lembaga pendidikan formal, maka hasil pembelajaran Coding Smart School harus relevan dengan kehidupan nyata, sesuai dengan kondisi kerja, dan kebutuhan pasar dan atau malah menciptakan lapangan kerja baru khususnya di bidang teknologi dan programming komputer.

Kurikulum Kemdikbud
Siswa akan mendapat pendidikan kurikulum Kemdikbud dengan mata pelajaran:

Matematika
Bahasa Indonesia
Bahasa Inggris
Pedidikan Kewarganegaraan
Fisika
Kimia
Biologi

Ketujuh pelajaran tersebut sudah diolah menjadi modul cetak yang berisi:
- Teori Pembelajaran
- Materi Online yang harus dipelajari untuk pengayaan pengetahuan
- Kuis (pilihan ganda, mengisi, dan essay)

Sedangkan untuk materi Coding Smart School, siswa mendapat teori dan praktek program-program coding dengan tujuan dapat membuat web dan aplikasi.

Kurikulum Coding:
HTML
CSS
Javascript
Database
PHP
Ruby
Design
Java

Kurikulum Coding ini bisa diubah sesuai dengan perkembangan teknologi dan kebutuhan pasar.
