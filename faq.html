Apa itu Coding Smart School?

Coding Smart School adalah lembaga pendidikan untuk generasi muda
(14 - 20 tahun) yang berfokus mengajarkan keterampilan programming secara lengkap untuk generasi muda Indonesia.

Mengapa Coding Smart School berfokus pada keterampilan coding

Keterampilan coding / programming adalah keterampilan yang wajib dimiliki setiap orang yang ingin menguasai teknologi di abad teknologi ini. Dengan menguasai keterampilan coding, maka lulusan Coding Smart School sudah lebih unggul dalam keterampilan teknologi dibanding lulusan SMA bahkan SMK biasa.

Karena berfokus pada keterampilan, Coding Smart School sangat concern pada tenaga pendidik. Para pendidik di sini adalah para founder langsung, yakni para pemuda Indonesia yang sekarang menjadi expert di bidang-nya masing-masing, termasuk para CTO di berbagai startup dan scaleup di Indonesia dan Internasional.

Apa tujuan dari Program Coding Smart School?

1. Mendukung Program Pemerintah menghasilan 1000 developer Indonesia dalam 5 tahun.
2. Menghasilkan programmer generasi muda (14 - 20 tahun) yang sungguh-sungguh menguasai ilmu teknologi yang update dan mampu membuat aplikasi berbasis website, android, atau IOS, dan seterusnya.
3. Mendukung kaum perempuan untuk menjadi teknopreneur yang andal.

Bagaimana Metode Belajar di Coding Smart School?

Setiap hari, kami akan mengajarkan Kurikulum SMA + Materi Coding.
Dengan metode belajar kami yaitu:
60% Praktik dan 40% Teori.
Metode tersebut sudah terbukti efektif dan efisien bagi siswa untuk menerima dan menyerap pelajaran yang diajarkan para Founders.

Siswa seperti apakah yang diharapkan oleh Coding Smart School?

Siswa Coding Smart School diharapkan berkomitmen untuk mendukung Indonesia dalam bidang teknologi, selain itu Coding Smart School juga mengharapkan siswanya untuk mengaplikasikan keterampilan yang didapat untuk menciptakan solusi bagi masalah sehari-hari.

Apa saja manfaat dalam mengikuti Program Coding Smart School?

Memaksimalkan waktu dan hasil. Dalam 3 tahun belajar, siswa dapat menguasai coding dan mendapat bimbingan yang lengkap. Siswa juga belajar Kurikulum level SMA IPA dan akan mendapat Ijazah dari Dinas Pendidikan DKI Jakarta pada masa akhir inkubasi.

Apa saja persyaratan untuk mengikuti program Coding Smart School?

Persyaratan untuk mengikuti program Coding Smart School yaitu:
1. Siswa mempunyai passion untuk mendukung Indonesia dalam bidang teknologi.
2. Lulus dalam tahapan seleksi;
a. Seleksi tertulis pelajaran fisika, matematika, logika, bahasa Indonesia, bahasa Inggris
b. Seleksi kreatifitas yang didesain para founder Coding Smart School,
c. Seleksi wawancara
d. Seleksi beasiswa, bagi siswa yang benar-benar membutuhkan.
3. Jika terpilih menjadi siswa Coding Smart School, siswa harus berkomitmen untuk menyelesaikan program.

Adakah tahapan seleksi dalam menjadi siswa Coding Smart School?

Ada 4 tahapan seleksi:
a. Seleksi tertulis pelajaran fisika, matematika, logika, bahasa Indonesia, bahasa Inggris
b. Seleksi kreatifitas yang didesain para founder Coding Smart School,
c. Seleksi wawancara
d. Seleksi beasiswa, bagi siswa yang benar-benar membutuhkan

Apa yang harus siswa lakukan agar dapat lulus seleksi menjadi siswa Coding Smart School?

1. Siap test tertulis
2. Punya minat dan bakat untuk menjadi coder dan programmer
3. Dukungan orangtua

Bagaimana cara mendaftar seleksi Coding Smart School?

1. Pelajari Jadwal Seleksi
2. Siapkan Curriculum Vitae dan berkas yang diperlukan : raport, ijazah, KTP / KK
3. Mendaftar langsung, saat launching program
4. Mendaftar lewat email, di website Coding Smart School sebelum 30 November 2016

Siapa sajakah yang boleh menjadi siswa Coding Smart School?

1. WNI usia 14 - 20 tahun

Komitmen seperti apa yang diharapkan Coding Smart School pada siswanya?
